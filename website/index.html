Every style is just Markdown

Install in three lines

Open source, Swift 6, macOS 13+. Clone it, trust it, deploy it.

Terminal — fixit 
$ git clone https://github.com/example/fixit
$ make trust-signing
$ make deploy
# bonus: it's a CLI too
$ swift run Fixit --fix --style native \
    --text "lets create a new project on this folder"
  1. 1

    Grant Accessibility permission. It's only used to copy your selection and paste the result back, nothing else.

  2. 2

    Open Settings from the menu-bar icon.

  3. 3

    Paste your OpenRouter API key. It goes straight into the macOS Keychain.

That's it. Fixit lives in your menu bar and waits for the hotkey.
